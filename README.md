# Mini Integration Platform

A backend integration platform inspired by workflow automation tools such as Zapier.

## Planned Features

* Webhook Receiver
* Event Queue
* Background Workers
* Retry Handling
* External API Integrations
* OAuth Support
* Workflow Execution Engine

## Tech Stack

* Java
* Spring Boot
* PostgreSQL
* Redis
* Kafka

## Status

Work in Progress

This project is being developed as part of backend engineering and system design preparation.
